GLOBAL VISUAL RULES: The entire palette is constrained to a tight split-complementary harmony anchored on deep blue-slate. Base backgrounds use only #0a1118, #0f1923, #111a24, #141e2a, #1a2838. Borders use only #1e2d3a. Text hierarchy: #d4dce4 for primary values, #8a9aa8 for secondary text, #5a6a78 for labels and headers, #3a4a58 for disabled/muted elements. Accent colors are desaturated and muted — teal #4a9e8e (primary accent), green #3d8a5a (positive states), amber #b8923a (warning states), coral-red #c45a52 (critical/danger states). NO cyan #58a6ff, NO bright neons, NO saturated blues, NO purple, NO pink, NO magenta, NO bright white #ffffff. All whites are cool blue-tinted #d4dce4. Typography is primarily monospace (Fira Code, JetBrains Mono, Consolas) for ALL data values, table cells, and metrics — Inter or system sans-serif only for navigation labels and section headers. Every surface is fully opaque — zero backdrop-filter, zero blur, zero frosted glass, zero glassmorphism. Shadows are limited to a single 0 2px 4px rgba(0,0,0,0.3) on the top nav only — nowhere else. Border-radius is 0px everywhere except metric tiles (4px) and the avatar circle. The design philosophy is brutalist-utilitarian: if a visual element does not convey data, it does not exist. No decorative gradients, no shimmer animations, no pulse animations, no glow effects. The only motion implied is in the sparklines and chart data. 8K resolution, hyper-detailed interface design, production-ready precision.
