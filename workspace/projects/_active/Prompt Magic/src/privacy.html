✨ Prompt Magic — Privacy Policy

Last updated: April 28, 2026

TL;DR: Your data never touches our servers. Prompt Magic operates entirely between your browser and the AI provider you choose. We collect nothing.

1. What Prompt Magic Does

Prompt Magic is a Chrome extension that improves the quality of prompts you type into AI chat interfaces (such as Gemini, ChatGPT, and Claude). When you click the ✨ button, your text is sent directly from your browser to the AI provider's API — and the improved prompt is injected back into the chat box.

2. Data We Collect

None. Prompt Magic does not collect, transmit, store, or sell any personal data, browsing history, analytics, telemetry, or usage metrics. There is no backend server. There is no database. There are no cookies.

3. API Keys (BYOK)

Prompt Magic uses a Bring-Your-Own-Key (BYOK) model. You provide your own API key from a supported provider (Google, OpenAI, or Anthropic). Your key is:

  • Stored locally in your browser's chrome.storage.local
  • Never transmitted to any server other than the AI provider you selected
  • Never logged or cached outside your browser
  • Deletable at any time by clearing the extension's storage or uninstalling

4. Prompt Content

When you click ✨, your prompt text is sent directly from your browser to the selected AI provider's API endpoint (e.g., generativelanguage.googleapis.com). Prompt Magic acts as a pass-through — it does not intercept, log, copy, or store your prompts. The AI provider's own privacy policy governs how they process your input.

5. Permissions

The extension requests the following permissions:

  • activeTab — to detect the input field on the current page
  • storage — to persist your API key and model selection locally
  • host_permissions — to make API calls to Google, OpenAI, and Anthropic endpoints
  • sidePanel — to render the extension's settings panel

6. Third-Party Services

Prompt Magic communicates only with the AI provider you explicitly configure:

  • Google Gemini API — Terms of Service
  • OpenAI API — Terms of Use
  • Anthropic API — Terms of Service

7. Children's Privacy

Prompt Magic is not intended for use by individuals under the age of 13. We do not knowingly collect any information from children.

8. Changes to This Policy

If this policy is updated, the "Last updated" date at the top will reflect the change. Continued use of the extension after changes constitutes acceptance of the revised policy.

9. Contact

For questions or concerns about this privacy policy, contact: privacy@promptmagic.app
