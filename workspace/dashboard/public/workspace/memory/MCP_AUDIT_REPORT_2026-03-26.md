# MCP Audit & Strategy: The Imperial Expansion (v1.0)

## 1. Objective
Optimize the MCP server ecosystem to complement Google Antigravity (GaG) and offload specialized operations from the main reasoning loop.

## 2. Current State Assessment
- **Active Servers**: 11
- **Core Strengths**: Infrastructure (GCP, Supabase), Search (Firecrawl, Apify), Basic Dev (GitHub).
- **Critical Gaps**: 
    - No direct Email/Communication automation.
    - No specialized Code Quality/Security tools.
    - Weak Local Knowledge Management (RAG) beyond Pinecone.
    - No high-fidelity research tools (Perplexity disabled).

## 3. The "Imperial Guard" - Recommended MCP Expansion

### A. Communication & Workflow (Offloading Triage)
1. **Gmail/Google Workspace**: Full integration for `check_emails.py`.
2. **Slack/Discord**: Automated status reporting and escalation.
3. **Linear/Jira**: Syncing tasks from `night_watch.md` to formal project management.

### B. Specialized Research & Intelligence
4. **Perplexity (Re-activate)**: Real-time high-fidelity web search.
5. **Brave Search**: Fast, privacy-focused alternative for diverse results.
6. **Exa (formerly Metaphor)**: AI-native search for finding high-quality documentation and resources.

### C. Technical Deep-Dives (PHP/Legacy Refactoring)
7. **PHP/Larastan**: Specialized static analysis for NorCast refactoring.
8. **SQL/PostgreSQL**: Deep introspect for Supabase/Postgres beyond generic CRUD.
9. **Docker**: Direct container management for local testing.

### D. Memory & Knowledge (The "Imperial Library")
10. **Memory (MCP Implementation)**: A persistent, graph-based local memory for cross-task context.
11. **Everything (Windows Search)**: High-speed local file indexing for deep filesystem awareness.

## 4. Synergy with GaG
- **GaG (The Architect)**: Sets the strategy and high-level directives.
- **MCPs (The Specialized Tools)**: Provide high-fidelity sensors and actuators.
- **Cline (The Operator)**: Orchestrates the tools to fulfill the strategy.
