# 10 Best Practices

1. **Git Snapshots**: Take Git snapshots before every single turn.
2. **Terminal Whitelist**: Only allow `uv`, `npm`, `git`, and `python` in the terminal. No destructive commands like `rm -rf`.
3. **Execution Safety**: Do not auto-execute commands outside the whitelist.
4. **Symlink Skills**: Use Symbolic Links for the skills directory.
5. **Cost Defense**: Use MCP Vault (mcpv) with Smart Context Valve to block redundant scans.
6. **PR-First Workflow**: All tasks must follow the PR-First workflow (branch and PR, no direct pushes to main).
7. **Verification**: A task is only complete upon generation of a Browser Recording Artifact showing functionality.
8. **RBI Architecture**: Use Role, Background, Instruction for all task planning.
9. **Narrow Scope**: Operate strictly within the official Google Antigravity individual preview parameters in this directory only.
10. **Model Usage**: Use Gemini 3.1 Pro for planning, and Claude Opus 4.6 for heavy coding turns.
