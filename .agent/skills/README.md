# Skills Directory

This directory is configured to use Symbolic Links (Symlinks) for portability. 
You can symlink your central skill repositories (e.g., Antigravity Awesome Skills) into this folder.

Example:
```bash
# On Windows (Run as Admin or if Developer Mode is on)
mklink /D "c:\Users\steph\Desktop\Antigravity & Agent 0\.agent\skills\awesome-skills" "C:\path\to\antigravity-awesome-skills"
```
